Varsinaiset testit on luotu kahteen eri hakemistoon. Savutestit, integraatiotestit sekä järjestelmätestit löytyvät hakemistosta <b>keeptrack-ts/demo</b>, tiedostoista <b>smoketests.robot, integrationtests.robot</b> sekä <b>systemtests.robot</b>. Näiden lisäksi yksikkötestit on luotu hakemistoon <b>keeptrack-ts/src/projects</b>, jossa valmiina oleviin tiedostoihin <b>ProjectsPage-test.tsx, ProjectList-test.tsx, ProjectForm-test.tsx</b> sekä <b>ProjectCard-test.tsx</b> kuhunkin on lisätty yksi uusi yksikkötesti, jonka lisäksi on luotu samaan hakemistoon uusi testitiedosto <b>projectDetail-test.tsx</b> kyseisen komponentin testaukseen.
